# FortSource
A Recreation of OG Fortnite's Movement in UE5.4.4

Current Features:
- Basic Locomotion with Start and Stop Animations.
- Additve Animations.
- Replicated Sprinting.

Planned Features:
- Crouching.
- Turn in Place.
- Crouch Sprinting.
- Pivot.
- Jump Up, Apex, Fall and Land Animation.
- Replicated Aim Offset.
